ROSTER
근무표 당직표 도움말
  1. 화면 상의 모든 칸은 텍스트 필드로 직접 입력이 가능합니다.
  2. 근무표 생성하기: 먼저 배치시킬 근무자의 이름을 입력하고 각 근무자들의 근무수를 입력한 뒤 수동으로 입력할 근무자들을 직접 근무표에 입력한 후 'make roster' 버튼을 클릭하면 근무표가 생성됩니다.
      *주의사항*
    1. 수동으로 직접 입력이 필요한 근무자들은 근무표의 해당 칸을 클릭하여 입력하면 됩니다. 그 후에 'make roster'를 누르시면 이미 입력된 칸은 유지시킨채 나머지 조건에 맞춰 생성을 완료합니다.
    2. 만약 수동으로 입력한 근무자의 근무수와 명단에서 근무자들에게 배정한 근무수의 총합이 28이 아닐 경우 근무수 칸이 붉은색으로 변합니다.
    3. '근무수 채우기' 버튼을 누르시면 현재 근무표에 배정된 인원과 더 필요한 근무수를 다 계산하여 자동으로 근무수를 배정해줍니다. 수동으로 직접 근무표를 입력하고 나면 이 버튼을 사용하시길 권장드립니다.
      다만, 근무수가 나누어 떨어지지 않을 경우 밑에서부터 1번 더 들어가 는 방식이기에 형평성에 따라 클릭 후 세부적으로 수정하시면서 녹색을 유지하면 됩니다.
    4. 근무표 지우기 1 버튼은 근무표 전체를 지우고 / 근무표 지우기 2는 make roster 를 누르기 직전의 근무표 상태로 되돌려줍니다.
    5. 근무 열외 요일은 해당 근무자를 체크한 요일의 근무에서 열외시키는 기능을 수행합니다.
    6. 근무 열외 요일 끝에 있는 초기화버튼은 해당 근무자의 근무수와 근무열외 요일을 모두 지워줍니다.
    7. 중복 허용버튼은 근무자가 7명 미만일때 사용하기 위한 것입니다. 먼저 중복 허용에 체크를 하고 몇번의 중복을 허용할 것인지를 선택하면 됩니다. 근무자 수가 적을 수록 많은 중복을 허용해야 근무표 생성이 가능해집니다.
    8. 근무자 이름을 기입할 때 꼭 위에서부터 차례로 기입할 필요는 없습니다. 아무 칸에 아무 순서로 넣어도 '근무수 채우기' 버튼이나 근무수를 계산하는 것에 지장을 주지 않습니다.
